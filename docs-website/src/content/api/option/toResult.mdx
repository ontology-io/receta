# option/toResult

## Functions

### toResult()

#### Call Signature
 `>` **toResult**\<`T`, `E`\>(`option`, `error`): [`Result`](../result/types.md#result)\<`T`, `E`\>

Defined in: [option/toResult/index.ts:39](https://github.com/maxios/receta/blob/cc60b386f67b185c095dab8e7dc7a371fe636c2d/src/option/toResult/index.ts#L39)

Converts an Option to a Result.

If the Option is Some, returns Ok with the value.
If the Option is None, returns Err with the provided error.

##### Type Parameters

###### T

`T`

###### E

`E`

##### Parameters

###### option

[`Option`](types.md#option)\<`T`\>

The Option to convert

###### error

`E`

Error value to use if None

##### Returns

[`Result`](../result/types.md#result)\<`T`, `E`\>

Result containing the value or error

##### Example

```typescript
// Data-first
toResult(some(42), 'No value') // => Ok(42)
toResult(none(), 'No value') // => Err('No value')

// Data-last (in pipe)
pipe(
  findUser(id),
  toResult({ code: 'USER_NOT_FOUND', message: 'User not found' }),
  flatMap(user => validateUser(user))
)

// With error function
pipe(
  config.get('apiKey'),
  toResult('Missing API key in config')
)
```
