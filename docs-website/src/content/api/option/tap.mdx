### tap()

#### Call Signature
 `>` **tap**\<`T`\>(`option`, `fn`): [`Option`](types.md#option)\<`T`\>

Defined in: [option/tap/index.ts:35](https://github.com/maxios/receta/blob/cc60b386f67b185c095dab8e7dc7a371fe636c2d/src/option/tap/index.ts#L35)

Performs a side effect on Some value without modifying the Option.

If the Option is Some, calls the function with the value.
Returns the original Option unchanged.

Useful for logging, debugging, or other side effects.

##### Type Parameters

###### T

`T`

##### Parameters

###### option

[`Option`](types.md#option)\<`T`\>

The Option to tap

###### fn

(`value`) => `void`

Side effect function

##### Returns

[`Option`](types.md#option)\<`T`\>

The original Option

##### Example

```typescript
// Data-first
tap(some(42), x => console.log('Value:', x))
// Logs: "Value: 42"
// => Some(42)

// Data-last (in pipe)
pipe(
  findUser(id),
  tap(user => console.log('Found user:', user.name)),
  map(user => user.email)
)
```
