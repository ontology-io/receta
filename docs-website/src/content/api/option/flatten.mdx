##### Example

```typescript
// Data-first
flatten(some(some(42))) // => Some(42)
flatten(some(none())) // => None
flatten(none()) // => None

// Data-last (in pipe)
pipe(
  some(some(5)),
  flatten,
  map(x => x * 2)
) // => Some(10)
