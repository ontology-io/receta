# collection/groupByPath

## Functions

### groupByPath()

#### Call Signature
 `>` **groupByPath**\<`T`\>(`items`, `path`): `Record`\<`string`, `T`[]\>

Defined in: [collection/groupByPath/index.ts:44](https://github.com/maxios/receta/blob/cc60b386f67b185c095dab8e7dc7a371fe636c2d/src/collection/groupByPath/index.ts#L44)

Groups items by a nested object path.

Useful for grouping objects by nested properties like `user.role` or `metadata.category`.

Uses type-safe path access via Option pattern. Items with undefined/null paths
are grouped under the 'undefined' key.

##### Type Parameters

###### T

`T`

##### Parameters

###### items

readonly `T`[]

The array of items to group

###### path

`string`

Dot-separated path to the property (e.g., 'user.role')

##### Returns

`Record`\<`string`, `T`[]\>

A record mapping path values to arrays of items

##### Example

```typescript
// Data-first
const users = [
  { name: 'Alice', profile: { role: 'admin' } },
  { name: 'Bob', profile: { role: 'user' } },
  { name: 'Charlie', profile: { role: 'admin' } }
]

groupByPath(users, 'profile.role')
// => {
//   admin: [
//     { name: 'Alice', profile: { role: 'admin' } },
//     { name: 'Charlie', profile: { role: 'admin' } }
//   ],
//   user: [{ name: 'Bob', profile: { role: 'user' } }]
// }

// Data-last (in pipe)
pipe(
  users,
  groupByPath('profile.role')
)
```
