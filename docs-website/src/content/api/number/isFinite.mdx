Type guard to check if a value is a finite number.

Returns true if the value is a finite number (not Infinity, -Infinity, or NaN).

#### Parameters

##### value

`number`

The value to check

#### Returns

`boolean`

True if the value is finite

#### Example

```typescript
isFinite(42) // => true
isFinite(42.5) // => true
isFinite(Infinity) // => false
isFinite(-Infinity) // => false
isFinite(NaN) // => false
```
